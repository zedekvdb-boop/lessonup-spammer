LessonUp.app Autojoiner

This is a simple tool for automatically joining a [suspicious link removed] session with multiple bot users. You can specify a custom name and the number of bots you'd like to send.

 How to Use (for Users)

Go to the Releases page on the right.

Download the latest lessonup_bot_global.exe file.

Run the .exe file. A terminal window will open.

Enter the LessonUp code when prompted.

Enter the base name you want to use for your bots (e.g., joey).

Enter the number of bots you want to send.

Watch the script initialize and send all your bots to the lesson!

Requirements

Windows 10 or 11.

Google Chrome or Chromium must be installed. The script will try to find Chromium first, and if it can't, it will use Google Chrome as a fallback.
